# ClimatePolicyDiffusionParadox
Files to reproduce results from the associated pape titled "The paradox of climate policy diffusion"

This repository contains the main R markdown file to reproduce the results as well as the data necessary to run the code.

Before running the R code, please make sure to install all the associated packages, save the data files on your computer and update the working directory in the code.
